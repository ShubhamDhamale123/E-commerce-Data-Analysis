# E-commerce-Data-Analysis
E-commerce Data Download from Kaggle that throught i have created interactive dashboard using multiple visual ,it's used  easy to understanding the client.


◆ I excite to share my E-commerce Sales Analysis Dashboard .
Built an interactive dashboarad using Text, Card, Slicer, Slicer Button, Pie Chart, Tree Map, Bar Chart And Line chart etc., That help to easy to undestand the E-commerce sales .

🎯 Key Highlight :
1) Category Wise Sales : Understand the which category done the most sales.
2) Payment mode : Shows the different type of payment mode,which category sale did the top most payment on the COD, Credit Card, Debit Card,
EMI, UPI etc.,
3) Sub-Category Wise Sales : Understand the which sub-category done the most sales.
4) Amount vs Sold Amount : Shows the different between the Amount and sold amount using the card.
